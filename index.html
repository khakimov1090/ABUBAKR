PIFAGOR TEOREMASI

Pifagor teoremasi qadimgi matematikadan kelib chiqqan va turli xalqlar tomonidan mustaqil ravishda kashf etilgan. Uning eng mashhur shakli qadimgi yunon matematiklari bilan bog'liq, xususan Pifagor (miloddan avvalgi taxminan 570–495-yillar) bilan. Quyida teoremaning kelib chiqish tarixiga qisqacha to'xtalamiz:

Pifagor uchburchagi yoki to'g'ri burchakli uchburchak – bu uchburchakning ikkita oyoqlari (katetlari) kvadratlarining yig'indisi uchinchi tomonning (gipotenuzaning) kvadratiga teng bo'ladigan uchburchakdir. Bu munosabat Pifagor teoremasi bilan ifodalanadi:

a² + b² = c²

Bu yerda: - a va b – uchburchakning katetlari, - c – gipotenuza.

Pifagor Uchburchagiga Mos Uchburchaklar Pifagor uchburchagi hosil qiluvchi butun sonlar to'plamiga Pifagor sonlari deyiladi. Masalan:

1. 3, 4, 5: 3² + 4² = 5² 9 + 16 = 25

2. 5, 12, 13: 5² + 12² = 13² 25 + 144 = 169

3. 8, 15, 17: 8² + 15² = 17² 64 + 225 = 289

Uyda Pifagor Teoremasining Qo'llanilish Joylari:

1. TOM QURILISHI (Gipotenuza Hisoblash) Tomning qiyalik burchagini va uzunligini hisoblashda ishlatiladi.

2. NARVON UZUNLIGI Narvon uy devoriga tik turganda, uning uzunligini aniqlash uchun Pifagor teoremasi ishlatiladi.

3. POL YUZALARI (Diagonalni Topish) Pol yoki shiftning diagonal uzunligi hisoblanadi.

4. DEVORLARDA QUVURLAR VA KABEL YO'NALISHLARI Elektr simlari yoki suv quvurlari diagonal bo'ylab o'tkazilishi kerak bo'lganda.

5. LANDSHAFT LOYIHALASH Uy atrofidagi yo'laklar yoki bo'laklarning to'g'ri burchakli joylashuvini belgilash uchun.
